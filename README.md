# User Data Analysis Project (Python)

This repository contains a step-by-step user data analysis project built to
practice core data science concepts using Python.

The project works with a social-network-style dataset containing:
- users
- their connections (friends)
- pages they have interacted with

The goal is to understand how raw, nested data is transformed into clean,
analysis-ready data and then used to build simple recommendation features.
